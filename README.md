# JanSaarthi - Rock-AI-For-Bharat

JanSaarthi is an AI-powered platform that helps citizens discover government schemes they are eligible for. It uses rule-based matching for accuracy and AI to extract details from ID scans and simplify complex scheme information into clear, easy-to-understand explanations in local languages.

## Run Locally

**Prerequisites:** Node.js

1.  **Install dependencies:**
    ```bash
    npm install
    ```
2.  **Set Environment Variables:**
    Set the relevant keys (like `VITE_GEMINI_API_KEY`) in your `.env.local` file.
3.  **Run the app:**
    ```bash
    npm run dev
    ```
